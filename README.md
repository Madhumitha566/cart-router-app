> A responsive single-page application built with **ReactJS** and **React Router** that demonstrates key e-commerce functionality: product listing, stateful cart management, and dynamic pricing calculations.

## ‚ú® Features

This project fulfills all the specified requirements by implementing the following features:

* **Product Listing:** Fetches and displays a full catalog of products from the **Fake Store API** on the main page (`/`).
* **Responsive Layout:** Products are displayed in a responsive, user-friendly grid format.
* **Cart Functionality:**
    * **Add/Remove Items:** A button toggles functionality to add a product to the cart or remove it if already present.
    * **Quantity Control:** Users can easily increase or decrease the quantity of any item directly in the cart.
* **Dynamic Pricing:**
    * Calculates the subtotal for each item (`Price * Quantity`).
    * Displays the overall **Total Cart Price** (Subtotal of all items).
    * Applies a **10% discount** to the final price.
* **Routing:** Uses `react-router-dom` to ensure clean navigation between the **Product Page** and the **Cart Page**.
